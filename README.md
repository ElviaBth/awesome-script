# Awesome-Script
This script is to calculate the percentage of Decidim installations that have DecidimAwesome installed.

## Gems used
- Faraday
- Nokogiri
- Open-uri
- Dotenv/load
- JSON
  
## API Info
All data is extracted from an AirTable API.
